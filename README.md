# Exercícios de Lógica com JavaScript 🚀

Este repositório contém a resolução de 5 problemas de lógica de programação utilizando **JavaScript** e estruturas de repetição (`while` e `do...while`).

## 📋 Descrição das Atividades

1.  **Validação de Positivos:** O programa solicita um número e só encerra quando o valor for positivo.
2.  **Média de Notas:** Recebe notas indefinidamente (0 a 10). O programa para ao receber um número negativo e calcula a média das notas válidas.
3.  **Totalizador de Compras:** Soma valores de produtos inseridos pelo usuário até que o valor `0` seja digitado.
4.  **Validação de Nome:** Garante que os campos de "Nome" e "Sobrenome" não fiquem vazios.
5.  **Sistema de Saque:** Simula uma conta bancária onde o usuário só pode sacar valores disponíveis em saldo até que este chegue a zero.

## 🛠️ Tecnologias Utilizadas

- **JavaScript (Node.js)**
- **readline-sync:** Biblioteca para captura de inputs no terminal.
- **Git/GitHub:** Para versionamento de código.
